## 1 任务概述

1. 车牌定位：输入车辆图片后，输出mask图以及定位到的车牌图片；
2. 字符分割：根据获得的车牌图片进行字符分割，输出分割后单个字符；
3. 字符识别：在给定模板下对分割出的字符进行识别。
